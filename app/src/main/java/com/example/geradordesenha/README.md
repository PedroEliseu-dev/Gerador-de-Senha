# Gerador de Senha - Android

Aplicativo Android desenvolvido em Kotlin usando Jetpack Compose para gerar senhas seguras e aleatórias.

## Funcionalidades

- Gerar senha aleatória
- Copiar senha para área de transferência
- Interface moderna com Material 3
- Indicador de força da senha

## Tecnologias utilizadas

- Kotlin
- Jetpack Compose
- Material 3
- Android Studio
